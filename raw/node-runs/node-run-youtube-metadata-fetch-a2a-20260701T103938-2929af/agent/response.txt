## Inputs

- `source_url` - required YouTube watch or youtu.be URL.
- `api_url` - optional API base URL. Defaults to `YOUTUBE_CONTENT_API_URL`, then the managed YouTube content API endpoint.
- `api_token` - optional bearer token. Defaults to `YOUTUBE_CONTENT_API_TOKEN`, then `YOUTUBE_RESEARCH_WORKFLOW_TOKEN`.
- `timeout_seconds` - optional HTTP timeout. Defaults to `30`.
- `output_dir` - optional directory for generated files.
- `output_json` - optional explicit JSON output path.

## Outputs

The command returns normalized metadata as JSON and can write these files when
`--output-dir` is provided:

- `metadata.json`
- `source-url.txt`
- `result.json`

The normalized JSON includes:

- `video_id`
- `title`
- `author`
- `published_at`
- `youtube_url`
- `thumbnail`
- `source_url`
- `fetched_at`
